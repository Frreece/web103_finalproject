# Wireframes

Reference the Creating an Entity Relationship Diagram final project guide in the course portal for more information about how to complete this deliverable.

## List of Pages

- ⭐ Sign In
- ⭐ My Requests - Tenant
- ⭐ New Request - Tenant
- ⭐ Request Details - Tenant
- ⭐ Assigned Work - Technician
- Manager All Requests
- Manager Assign Technician
- Manager Dashboard
- Resolved Request History - Tenant

## Wireframe 1: Sign In

The Sign In page is the entry point for LeaseTrack. Users will sign in and access the app based on their role as a tenant, manager, or technician.

![Sign In Wireframe](https://i.imgur.com/Tq4BhiD.png)

## Wireframe 2: My Requests - Tenant

The My Requests page allows tenants to view their submitted maintenance requests, check request statuses, and navigate to individual request details.

![My Requests Tenant Wireframe](https://i.imgur.com/0JjSEfo.png)

## Wireframe 3: New Request - Tenant

The New Request page allows tenants to create a new maintenance request by selecting their building, unit, issue category, urgency level, and describing the issue.

![New Request Tenant Wireframe](https://i.imgur.com/urHtT2m.png)

## Wireframe 4: Request Details - Tenant

The Request Details page allows tenants to view the full details of a maintenance request, including status, category, urgency, assigned technician, activity updates, comments, and resolution notes.

![Request Details Tenant Wireframe](https://i.imgur.com/OJJxcoQ.png)

## Wireframe 5: Assigned Work - Technician

The Assigned Work page allows technicians to view the maintenance requests assigned to them and update the status of their work.

![Assigned Work Technician Wireframe](https://i.imgur.com/AYBjmZq.png)
